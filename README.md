# ⚡ Система оптимизации распределения электрической энергии

Проект для решения задачи потокораспределения в электрической сети «Альфа».  
Находит оптимальный план доставки энергии от источников к потребителям с учётом пропускной способности рёбер.

## Принцип работы

Проект реализует двухэтапный подход к решению задачи потокораспределения:

### 1. ML-приближение (нейросеть)
- Генерируются синтетические данные с помощью Latin Hypercube Sampling
- Обучается нейросеть `PathWeightNetwork` для предсказания весов распределения потоков по путям
- Модель использует фиктивный путь для учёта потерь энергии
- Вход: вектор признаков (пропускные способности рёбер + заявки)
- Выход: веса распределения потоков по всем возможным путям

### 2. Точный расчёт (солвер)
- Градиентный спуск с двухэтапным обновлением:
  - **Этап 1**: корректировка суммарного потока по заявке
  - **Этап 2**: перераспределение между путями
- Функция потерь: недопоставка + штраф за превышение пропускной способности
- Может использовать ML-приближение как начальную точку

### Правила распределения
- При превышении пропускной способности ребра — потоки перенаправляются по другим путям
- При невозможности полной доставки — ограничение пропорционально заявленным объёмам

## Входные данные

### `edges.json`
Список рёбер графа с пропускной способностью:
```json
[
  {"nodes": ["P", "XI."], "capacity": 2670.3},
  {"nodes": ["37", "VIII."], "capacity": "inf"}
]

```

* "inf" — неограниченная пропускная способность

### `flows.json`
Заявки от источников к потребителям:

```json
{
  "A": {
    "1": 738.0,
    "3": 1200.0,
    "4": 420.0
  },
  "B": {
    "1": 1074.7,
    "3": 403.387
  }
}
```

* Ключи верхнего уровня — источники (A-Z)

* Ключи второго уровня — потребители

* Значения — требуемая мощность в кВт

## Интерактивный граф

При наведении на элементы графа отображается подробная информация. Пример:

### Узел-источник

```text
Узел: G
Тип: source

Доставка потребителям:
  1: 911.0 / 911.0 | 100.0%
  3: 117.0 / 117.0 | 100.0%
  ...
  -----------------------------
  ИТОГО: 3583.3 / 3740.0 | 95.8%
```

### Узел-потребитель
```text
Узел: 1
Тип: consumer

Получено от источников:
  A: 738.0 / 738.0 | 100.0%
  B: 1004.4 / 1074.7 | 93.5%
  ...
  -----------------------------
  ИТОГО: 15123.5 / 15800.0 | 95.7%
```

### Ребро
```text
Ребро: v4 ↔ V.
Направление: v4 → V.
────────────────────────
Поток v4→V.: 3327.97 кВт
Поток V.→v4: 1146.98 кВт
────────────────────────
Суммарный поток: 4474.95 кВт
Максимум: 5034.8 кВт
Загрузка: 88.9%
```
